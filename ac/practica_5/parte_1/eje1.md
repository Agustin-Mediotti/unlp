# Funciones de las etapas del cauce
Indicar a qué etapa corresponde cada función que realiza el procesador al ejecutar una instrucción típica:

A. Decidir si se toma un salto o no => 2. **ID: Decodificación de Instrucción** 
B. Buscar la instrucción en memoria y llevarla a la CPU => 1. **IF: Obtención de Instrucción**
C. Calcular la dirección de un acceso a memoria => 3. **EX: Ejecución principal de instrucción**
D. Guardar el valor de un registro en memoria => 4. **MEM: Acceso a Memoria** 
E. Traer de memoria un valor a un registro intermedio => 4 **MEM: Acceso a Memoria** 
F. Almacenar el valor final de un registro => 5. **WB: Escritura de resultado en registro destino**
G. Calcular la dirección de un salto => 2. **ID: Decodificación de Instrucción** 
H. Verificar si están disponibles los operandos necesarios para continuar con la ejecución de la instrucción => 2. **ID: Decodificación de Instrucción**
